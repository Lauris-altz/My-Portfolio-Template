Descripción

Portfolio minimalista de diseño gráfico que muestra proyectos de branding, ilustración y diseño web. 
Diseñado para destacar trabajos creativos con navegación fluida, animaciones sutiles y contacto directo.

Objetivo

El objetivo de mi portfolio es mostrar cómo pienso y trabajo como diseñadora gráfica, reflejando mi estilo, mi proceso creativo y el valor que hay detrás de cada proyecto.
